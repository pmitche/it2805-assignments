IT2805 - Web Technologies

I dette emnet er det fokus på World Wide Web som en plattform for interaktive applikasjoner, publisering av informasjon og sosiale tjenester. Utvikling av web-applikasjoner krever brei kunnskap om den underliggende teknologien, formater og standarder som World Wide Web bygger på og i dette emnet lærer du om den underliggende kommunikasjonsprotokollen HTTP, markup språk som HTML, XHTML og XML, språk for å spesifisere formattering og transformasjoner som CSS og XSLT, event-basert programmering på klient-siden og dokumentmodellen DOM, interaktiv grafikk og multimediainnhold på web. Emnet gir også en innføring i søkemotorer og webindekser og aspekter ved innholdsforvaltning som standardiserte beskrivelser av innhold og regelverk for opphavsrett.

TDT4136 - Introduction to Artificial Intelligence

The subject starts with a description of problem solving methods by means of heuristic search. Therafter, various knowledge representation languages and inference methods for automatic problem solving. Representation in form of predicate logic, frames and semantic nets are treated, and connected to the main forms of reasoning - especially rule based reasoning. Furthermore, architectures that integrates various resoning methods, agent based architectures and architectures for interactive problem solving. Numerous applicaton examples are given to demonstrate the methods.

TDT4290 - Customer Driven Project

The course is accomplished in project groups. Each group is assigned a task specified by a customer. Therefore, the project is very close to what you could expect in a normal working life situation. All the phases of a typical IS/IT project are to be covered; project planning, preliminary study, requirements specification, design, coding, and evaluation. However, the focus should be on the early phases, that is, project planning, preliminary study, and requirements specification. Each group is to create a project report and to have a presentation and demonstration of the final product for the customer while an external examiner is present.
